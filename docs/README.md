## Technology Stack

All features are powered by:
- **Claude 3.5 Sonnet** - Credibility analysis, chat
- **Gemini 2.5 Flash** - Summaries, fact-checking, author research
- **Google Search Grounding** - Real-time web research (built into Gemini)
